# rubber-duck-debugging-front
Le Rubber Duck Debugging est une technique de débogage où vous expliquez votre code, ligne par ligne, à un canard en caoutchouc (ou tout autre objet inanimé). En verbalisant votre problème, vous prenez du recul et trouvez souvent la solution par vous-même. Ce site web rend cette technique accessible à tous, en ligne et de manière ludique !
